# Fenworks-Drone-Tracker
An parsing app used to make getting weekly drone forfeit information easier from the Fenworks website :)



Where do I copy?:

Copy info specifically starting with the "Week x start date:" 
This will be true for all different modes of drones, you will always start the copy from the same place.

You will not need to copy the season leaderboard for INDIVIDUAL ONLY, as it does not matter for the information we need. And will mess up the numbers if you do.

Email Tab: 

Once you have parsed all 3 game modes (make sure to do all of high school or all of college first, don't mix... will fix later to do and save both at the same time) head over to the email tab.

1. Select a school
2. Preview Email (see what it will look like in page) 
3. Add sub-bullet points for each stat in the email if you need to add more detailed info on a team in one of the 3 game modes. 
4. Open in Outlook. This will open the email in outlook so you can give it one final review before hitting send template ready to go specified to a specific school with their forfeit rates imported automatically. 
5. The stats for all schools Ind, Duos, and Combat all in one collective place.
